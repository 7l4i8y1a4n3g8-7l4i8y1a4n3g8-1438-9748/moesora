## Moesora 1.0.1

二次元少女风格的 Halo 2.x 博客主题，首个公开发布版本 (｡･ω･｡)ﾉ♡
主色为樱花粉 `#f9a8d4`，可在后台自由更换。

### ✨ 功能特性

- 二次元少女风格设计，主题色可自定义；浅色 / 深色 / 跟随系统三种配色，带切换与记忆
- 完整 SEO：动态标题、描述、Open Graph、Twitter Card、JSON-LD、RSS 自动发现
- 可选 Pjax 无刷新加载
- 自包含图片灯箱（零依赖、不走 CDN）：缩放 / 旋转 / 拖动 / 长图 / 双指 / 键盘
- 高级代码块：语言标识、行号、折叠、一键复制（兼容非 HTTPS 复制回退）
- 文章目录 TOC + 滚动高亮；CJK 感知字数统计与阅读时长
- 评论：内置评论 / Twikoo / Waline，客户端已内置打包，无需额外 CDN
- 相册：按后台分组以彩色标签切换展示，配合灯箱查看
- 瞬间：彩色时间轴布局，可点赞、可展开评论气泡
- 文章「赞赏支持」按钮兼作点赞：展开收款码的同时给文章点赞 +1
- 侧边栏布局可选：三列 / 仅左 / 仅右
- 正文视频 / 音频增强（mp4 / m3u8）、KaTeX 公式（深色适配）
- 可选 PWA、打印样式、站外链接跳转确认、客户端随机几何头像
- 留言墙、恋爱墙等自定义页面

### 📦 安装

下载下方的 `moesora.zip`，进入 Halo 后台 `主题 → 主题管理 → 安装主题`，上传该压缩包并启用即可。
更新时下载新版 `moesora.zip`，通过 `主题 → 升级` 上传安装包。

### ✅ 适配

Halo 2.14.0 及以上版本。
